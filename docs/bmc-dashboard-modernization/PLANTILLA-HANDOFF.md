# Plantilla de handoff — Solution ↔ Coding

**Propósito:** Formato estándar para "Solution entrega X → Coding puede empezar Y". Consistencia en los planes.

---

## Formato

```
## Handoff: [Solution/Coding] → [Coding/Solution]

**Entrega:** [nombre del entregable]
**Contenido:** [breve descripción]
**Ubicación:** [archivo, doc, o artefacto]
**Criterio de aceptación:** [qué debe cumplir para que el receptor pueda empezar]

**Receptor puede empezar:** [tarea Y]
```

---

## Ejemplo

```
## Handoff: Solution → Coding

**Entrega:** IA aprobada para Cotizaciones
**Contenido:** Flujo de estados, secciones, navegación
**Ubicación:** docs/bmc-dashboard-modernization/IA.md (aprobado)
**Criterio de aceptación:** Matias firmó; sin cambios pendientes

**Receptor puede empezar:** Implementar nav y wire /api/cotizaciones
```
