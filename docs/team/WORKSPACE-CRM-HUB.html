Cómo ver los Markdown con esta misma interfaz
  1. Servidor recomendado (enlaces ../ entre carpetas): en la raíz del repo: npm run team:hub (equivale a npx serve docs -l 4710) y abrí http://localhost:4710/team/WORKSPACE-CRM-HUB.html. Así los enlaces tipo ../AGENTS.md dentro de los Markdown resuelven bien.
  2. Solo carpeta team: npx --yes serve docs/team -l 4710 → http://localhost:4710/WORKSPACE-CRM-HUB.html (útil para el hub; los ../ hacia otros docs pueden fallar).
  3. No uses file://: el navegador bloquea la carga de archivos.
  4. Vercel: Simple Browser → https://calculadora-bmc.vercel.app/ (no se puede incrustar con la misma piel por políticas del sitio).
