Este cambio registra el commit `fix(deps): move playwright to devDependencies — breaks Alpine Docker build` dentro del sistema de trazabilidad del proyecto. Se modificó 1 archivo: package.json.

Contexto del commit:
playwright in production deps caused npm ci --omit=dev to install it in
the Alpine container. Playwright's native bindings require glibc (not musl),
causing server startup failure on Cloud Run.

Moving to devDependencies: excluded from docker builds (--omit=dev).
The /api/pdf/generate route handles ModuleNotFoundError gracefully (503)
and clients fall back to html2pdf.js raster PDF.
