## Descripción
Este cambio registra el commit `fix(pdf): replace fs.promises.glob with readdirSync — Node 20 compat` dentro del sistema de trazabilidad del proyecto. Se modificaron 2 archivos: server/routes/pdf.js, src/utils/quotationViews.js.

Contexto del commit:
fs.promises.glob was added in Node.js 22. server/Dockerfile uses node:20-alpine.
Static import of {glob} from "node:fs/promises" threw SyntaxError at module
load time, crashing server/index.js before it could listen on PORT — causing
every Cloud Run deployment to fail health checks since this route was added.

Replaced with readdirSync loop (Node 20 compatible). findChromiumSync is now
synchronous since there's no async I/O needed for path probing.
