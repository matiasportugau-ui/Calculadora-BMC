PlantaBordesEdgeStrips now divides each side into sub-strips based on
exterior geometry intervals (buildZoneBorderExteriorIntervals). When a
zone has a partial encounter on one side, the shared portion has no
strip (the clickable encounter line handles it); the free exterior
sub-portions each get their own strip independently selectable for
standard accessories. Fully exterior sides keep normal single-strip behavior.
