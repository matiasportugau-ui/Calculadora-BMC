# Checklist de cierre mensual — Facturación

**Propósito:** Pasos que administración debe seguir para cierre mensual. Orden y validaciones.

---

## Orden recomendado

1. **Exportar datos** — Facturas, notas de crédito, pagos del período
2. **Validar duplicados** — Revisar con Billing (billing-error-review)
3. **Conciliar** — Pagos vs facturas; montos
4. **Cerrar Pagos Pendientes** — Marcar período cerrado si aplica
5. **Archivar** — Backup o export para auditoría

---

## Validaciones

- [ ] No hay facturas duplicadas
- [ ] Notas de crédito aplicadas correctamente
- [ ] Estados de pago coherentes
- [ ] Fechas en rango del período
- [ ] Redondeos dentro de tolerancia
