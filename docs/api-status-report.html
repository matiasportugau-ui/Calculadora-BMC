Issues a resolver
🔴
CRITICO

Dashboard Sheets — 6 endpoints en 503

El error "Unable to parse range: 'Master_Cotizaciones'" indica que los tabs del spreadsheet BMC_SHEET_ID fueron renombrados o la service account perdió permiso. Los afectados son:

  • Master_Cotizaciones → /api/cotizaciones, /api/audit
  • Pagos_Pendientes → /api/kpi-financiero, /api/pagos-pendientes
  • BMC_VENTAS_SHEET_ID / BMC_STOCK_SHEET_ID → /api/ventas, /api/stock-kpi

Fix: abrir el spreadsheet en Google Sheets, verificar nombres de tabs, y re-compartir con la service account de GOOGLE_APPLICATION_CREDENTIALS.

🟡
MEDIO

Transportista — DATABASE_URL no configurado

El módulo de logística necesita PostgreSQL. DATABASE_URL no está en .env local. Los endpoints de trips, driver, y evidencias no funcionan en dev.

Fix: agregar DATABASE_URL=postgres://... al .env apuntando a la BD de staging/prod (o una local con docker run postgres).
