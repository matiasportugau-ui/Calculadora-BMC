🔗 Cómo funciona ngrok

🌐 Internet
Cliente / Webhook
→
☁️ ngrok
freewill-...ngrok-free.dev
→
💻 Tu Mac
localhost:80

1. Alguien hace una petición

Un navegador, una API externa (DGI, Google OAuth, Mercado Libre) o un webhook llama a:
https://freewill-kaleidoscopically-jacalyn.ngrok-free.dev

2. ngrok recibe la petición

Los servidores de ngrok (región South America) reciben el tráfico y lo enrutan por un túnel seguro hasta tu computadora.

3. Llega a tu app local

El proceso ngrok en tu Mac reenvía la petición a http://localhost:80. Tu servidor (Express, etc.) la procesa y responde.

4. La respuesta vuelve

Tu app → ngrok local → ngrok cloud → Internet. El cliente recibe la respuesta como si tu app estuviera en un servidor público.

⚠️ Tu app debe estar escuchando en el puerto 80. Si usas otro puerto (ej. 3001), ejecuta: ngrok http 3001
